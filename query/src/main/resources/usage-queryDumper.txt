Usage: java QueryDumper propertiesFile destinationFile
Example: java QueryDumper local.properties dump.txt

Properties file should include the following properties:
    querydumper.sql=select * from users \\ The sql query to be executed.
    querydumper.connection=name \\The name of the connection to be used.

Also database connection properties:
    database.name.def=Name
    database.name.class=...
    database.name.properties.database=(DESCRIPTION = (ADDRESS = (PROTOCOL= TCP) (Host= ...
    database.name.properties.user=...
    database.name.properties.password=...
